## 降噪：寻找 1Hz 脉搏

从 50,000,000 到 1。
没有任何冗余。

```verilog
// 核心逻辑
if (cnt == 26'd24_999_999) begin
    cnt <= 0;
    clk_1hz <= ~clk_1hz;
end
```
